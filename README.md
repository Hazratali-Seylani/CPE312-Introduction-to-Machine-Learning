# CPE312 - Introduction to Machine Learning
This repository contains practice codes and materials for "CPE312 - Introduction to Machine Learning" at Karabuk University in Turkey.

## Course Description
This course provides an introduction to the basics of machine learning, including:

- Supervised learning (classification and regression)
- Unsupervised learning (clustering and dimensionality reduction)
- Model evaluation and selection
- Feature selection and engineering
- Deep learning

Throughout the course, I will implement and practice machine learning algorithms and models using Python and popular libraries such as scikit-learn and TensorFlow week by week.
